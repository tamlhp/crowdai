This template is to mark your own images released into the Public Domain ("PD").
NB: Please do not use it for other people's images. Only use it if you, the uploader, are the sole copyright holder of the work and no works copyrighted by other people are included in it, unless those works may be used under an exception to copyright protections such as Freedom of panorama.

Additional information

The template is intended to be used in the following namespaces: the File namespace

The template is intended to be used by the following user groups: all users

Relies on:
{{PD-Layout}}

Localization

This template is localized by utilizing MediaWiki namespace messages. These can be translated at translatewiki.net (current translations).

The following: Wm-license-pd-author-with-author-text, wm-license-pd-author-with-author-any-purpose, Wm-license-pd-author-self-text, Wm-license-pd-author-not-legally-possible, wm-license-pd-author-self-any-purpose.


Examples

{{PD-self |author=[[:en:John Doe|John Doe]]}} gives

Please note that uploads to Commons must comply with
  1. Commons:Licensing: the file must be freely licensed, or public domain in both the source country and the United States;
  2. Commons:Photographs of identifiable people: for photographs taken in a so-called private place, any identifiable people in them must consent to publication; for some countries and situations this is also true for photographs taken in public (see Commons:Country specific consent requirements).
Files not meeting these criteria may be deleted. If in doubt, please ask for help, if possible before uploading.

Upload Wizard

The Upload Wizard is the default method for uploading files to Wikimedia Commons. You can select one or multiple files and begin uploading. Informations such as date and camera position are added automatically (if available in the photo's metadata).

Upload form

Where is the work from? (Click on the appropriate link)

  • It is from somewhere else
  • It is entirely my own work
  • It is someone else's work from Flickr (more information on uploading Flickr images)
  • It is from another Wikimedia project (Wikipedia, Wikibooks, Wikinews, etc.)
  • It is from a US federal government source
  • It is a free software screenshot
  • It is a non-free fair use image (Non-free logo, cover, screenshot, artwork etc.)
  • I don't know who the author is, or I don't know what license applies

Upload form for experienced users

  • Already know the license, and its copyright tag? Go directly to the main upload form or its experienced variant.
  • Already have an {{Information}} template ready? (For instance, generated by the Flinfo tool?) Use the basic upload form.

See also the full list of copyright tags for many nations. The first section of that page also has other general tags for public domain sources.

Please visit Commons:Help desk if you need to ask questions about uploading files.
Only upload freely licensed or public domain content. Fair use is not allowed on Wikimedia Commons. (help)

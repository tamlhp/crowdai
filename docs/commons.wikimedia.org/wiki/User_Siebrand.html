SieBot is my bot, but after 11 million or so edits, it is no longer active.

I used to run a Wikimedia wikis wide bot called CommonsDelinker, that delinked deleted images from Commons and can perform universal image replacement. It has since been rewritten and now has other maintainers and operators.

This is the root of the "license migration" template system. By sending it an appropriate keyword it allows GFDL images to be automatically retagged and sorted into one of several categories based on whether the file is eligible to participate in the licensing update effort.

This template is intended to be appended to {{GFDL}} and similar templates that indicate a possibly eligible licensing status. A parameter will be added to {{GFDL}} that passes through the keyword used here allowing for calls to GFDL to take the new form {{GFDL|migration=keyword}}.

Usage

{{License migration |1= }}

The action taken depends on the keyword specified:

  • Empty string or not recognized:
    An "announcement" highlighting the current relicensing effort is appended below the GFDL, and media files are added to Category:License migration candidates
If this file is eligible for relicensing, it may also be used under the Creative Commons Attribution-ShareAlike 3.0 license.

  • relicense:
    This keyword should be used when a media file does meet the relicensing criteria. The {{Cc-by-sa-3.0-migrated}} tag will be appended after the GFDL tag and the image will be categorized in Category:CC-BY-SA-3.0-migrated.
  • not-eligible:
    This keyword should be used if the media file does not meet the relicensing criteria. It removes the notice and categorizes the image in Category:License migration not eligible.
  • redundant:
    This keyword should be used if the media file is already dual licensed under CC-BY-3.0 or CC-BY-SA-3.0 and hence relicensing is not necessary. It removes the notice and categorizes the image in Category:License migration redundant.
  • needs-review:
    This keyword should be used if cursory inspection of the image cannot determine whether it is eligible for relicensing or not. These images are categorized for further attention in Category:License migration needs review and the announcement message remains in place.
  • opt-out:
    This keyword should only be used if the copyright holder for the image explicitly requests that they do not wish for this image to participate in the license migration. Doing this will remove the announcement and categorize the image in Category:License migration opt-out. The opt-out option will be available for a limited time only, and copyright holder wishing to exercise this option should act in an expeditious manner. For further discussion see Commons:License Migration Task Force/Migration#The opt-out provision.
